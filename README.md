# Консольная игра "Змейка" с системой рейтинга


Интерактивная игра, разработанная на Python. Проект демонстрирует работу с базами данных SQLite и асинхронную обработку ввода в реальном времени.

## 🛠 Стек технологий

* **Язык:** Python 3.x
* **База данных:** SQLite 3
* **Библиотеки:**
     * `keyboard` — для асинхронного управления.
    * `sqlite3` — встроенная работа с БД.
    * `re` — валидация ввода.

## 🚀 Быстрый старт

### Требования
Убедитесь, что у вас установлен Python 3.6 или выше.

# Управление


W, A, S, D — движение (вверх, влево, вниз, вправо).

ESC — экстренный выход в главное меню.

Enter — подтверждение выбора в меню.



# Архитектура проекта:


**Проект разделен на следующие модули:**

main.py — точка входа, главное меню и логика взаимодействия с пользователем.
game.py — игровой движок, обработка кадров (render) и физика столкновений.
entities.py — описание классов объектов (Snake, Apple).
database.py — слой доступа к данным, работа с SQL-запросами.

**Таблица лидеров**
В режиме "Рейтинг" программа сохраняет ваш результат. Вы всегда можете посмотреть ТОП-10 лучших игроков, выбрав соответствующий пункт в главном меню.
